Projectos

Aplicación de reconocimiento facial

Una aplicación web de pila completa que utiliza la pila (MERN), esta aplicación le permite identificar la detección de rostros en cualquier imagen suministrada. Esto se logró gracias a la API de inteligencia artificial de Clarifai. Utilicé React, Node, Express.js y MongoDB para crear esta aplicación web.

Clon de Twitter

Un clon simple y minimalista de Twitter, esta aplicación de pila completa le permite publicar tweets, contestar tweets, editar perfil, ver perfiles de usuarios, seguir o dejar de seguir y realizar las operaciones básicas de CRUD. Utilicé Node / Express para crear la API REST, autenticación de usuarios y MongoDB para el almacenamiento de datos. React, WebPack y Redux para el fronted.
